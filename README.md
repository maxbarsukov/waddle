## Building

### Pre-reqs

To build and run this app locally you will need a few things:

- Install [Node.js](https://nodejs.org/en/);

### Getting start

- Clone the repository
```bash
git clone --depth=1 https://github.com/maxbarsukov/waddle.git
```
- Install dependencies
```bash
cd waddle
npm install
```
- Run
```bash
npm run start
````
- **Tests**
```bash
npm test
```
- **Linting**
```bash
npm run lint
```
